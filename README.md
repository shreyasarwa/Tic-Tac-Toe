# Tic-Tac-Toe Web Game

A simple and interactive Tic-Tac-Toe game built using **HTML, CSS, and JavaScript**.  
This project allows two players to play against each other in the browser while tracking their scores.

---

## Features

- Two-player gameplay (Player 1 vs Player 2)
- Score tracking for both players
- First player to win **3 rounds** becomes the final winner
- Automatic board reset after each round
- Interactive game board
- Hover effects for better UI experience

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Game Rules

1. Player 1 uses **X** and Player 2 uses **O**.
2. Players take turns clicking the boxes.
3. The system checks for a winning combination after each move.
4. When a player wins a round:
   - Their score increases by **1**
   - The board resets automatically.
5. The first player to reach **3 wins** becomes the final winner.

---

## Winning Combinations

The game checks the following patterns:

- Horizontal rows
- Vertical columns
- Diagonal lines

Example:

X | X | X  
O | O |  
X |   |
